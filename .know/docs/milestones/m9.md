# M9 — 文档一致性机器检测

## 1. 目标

### 完成定义

`know-ctl check` 可检测模版与项目文档的结构偏差（section 缺失/多余/层级不匹配），输出具体不一致项

### 价值目标

用户可通过一条命令发现文档与模版的结构偏差，无需人工逐文件比对

### 关键假设

- **假设**: 从模版文件动态解析结构可以准确检测项目文档偏差
- **类型**: 技术可行性
- **验证方式**: 故意删除一个 section，运行 check，检查是否准确报告缺失

## 2. 计划

### 预期交付物

- 代码: know-ctl check 命令 + 模版解析逻辑
- 数据: 对 know 项目自身文档的 check 结果

### 达标标准

| 维度 | 标准 |
|------|------|
| 功能完整性 | 检测 3 种偏差类型：section 缺失/多余/层级不匹配 |
| 质量 | 真实偏差 0 漏检，误报率 0 |
| 性能与稳定性 | check 完整运行 <10s |
| 安全 | 本里程碑不涉及 |
| 可维护性 | 模版变更后 check 自动适配（动态解析，不硬编码） |
